### Step 2: Download the Application

On the Releases Page, look for the latest version of the application. There, you will find a downloadable file. Click on it to start the download.

### Step 3: Install the Application

Once the file has downloaded, locate it in your downloads folder. 

1. Extract the downloaded file to a folder of your choice.
2. Open your terminal or command prompt.
3. Change the directory to the extracted folder.
4. Run the command `composer install` to install necessary dependencies.

### Step 4: Set Up the Database

1. Create a new MySQL database for the application.
2. Open the `.env` file in the application folder.
3. Update the database connection details:

   ```
   DB_CONNECTION=mysql
   DB_HOST=127.0.0.1
   DB_PORT=3306
   DB_DATABASE=your_database_name
   DB_USERNAME=your_username
   DB_PASSWORD=your_password
   ```

### Step 5: Migrate the Database

In your terminal, run the command:

```
php artisan migrate
```

This command sets up the tables needed for the application.

### Step 6: Run the Application

Finally, run the application using:

```
php artisan serve
```

Open your web browser and visit `http://localhost:8000` to see the application in action.
